This script displays the classification results of different models over a 2D feature plane.
```bash

```

#### 2. Run Task 2 (Interactive 3D chart)
This script generates `iris_3d_boundary.html`; open it in your browser to interact with the 3D decision boundaries.
```bash
python task2.py

This script opens a window to visualize the SVM RBF probability surface in 3D.

```

#### 4. Run Task 4 (Slicing animation)

## ðŸ“Š Task Details

### Task 1: 2D Classifier Comparison
- **Features**: Sepal Length, Sepal Width
- **Models**:
  - Logistic Regression (RBF features)
  - Gradient Boosting
  - Logistic Regression (Binned features)
  - Logistic Regression (Spline features)
- **Output**: Matplotlib static plots showing the decision regions produced by each model.

### Task 2: 3D Linear Decision Boundaries
- **Features**: Sepal Length, Sepal Width, Petal Length
- **Model**: Multinomial Logistic Regression
- **Visualization**: Built with Plotly; supports rotation and zoom. Shows linear separating planes between the three classes.

### Task 3: 3D Non-linear Decision Surface
- **Features**: Sepal Length, Petal Length
- **Classes**: Setosa vs Virginica
- **Model**: SVM with RBF kernel
- **Visualization**: 3D surface where the height represents the probability difference (P(Virginica) - P(Setosa)), illustrating the non-linear decision boundary.

### Task 4: 3D Decision Boundary Slicing
- **Features**: Sepal Length, Sepal Width, Petal Length (used as the slicing axis)
- **Visualization**: Animated GIF. Fix a series of values for the third feature (Petal Length) and observe how the decision boundaries on the first two-feature plane change, which helps understand model behavior in higher dimensions.
